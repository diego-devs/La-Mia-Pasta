# La Mia Pasta

Sitio web de La Mia Pasta construido con React + Vite.

Deploy en GitHub Pages:
https://diego-devs.github.io/La-Mia-Pasta/

## Desarrollo local

Requisitos:
- Node.js 20+
- npm

Comandos:
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Deploy

El sitio se publica automáticamente en GitHub Pages mediante el workflow:
- `.github/workflows/deploy-pages.yml`

URL pública:
https://diego-devs.github.io/La-Mia-Pasta/
